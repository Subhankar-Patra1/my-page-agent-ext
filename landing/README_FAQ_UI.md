# Oryonix AI - FAQ UI Selection

The current FAQ interface implemented on the landing page is a **Hybrid Modern Grid Layout**.

## UI Classification
This design is a combination of two premium options originally suggested:

1. **Option 3: The "Bento Grid" Accordion**
   - It uses a multi-column grid system (3 columns on desktop).
   - It treats each FAQ as an independent module.
2. **Option 1: The Minimalist Concept**
   - It uses clean, dark-themed cards with subtle 1px borders.
   - It focuses on high-contrast typography and smooth animations.

## Key Features
- **Dynamic Grid**: Automatically adjusts from 3 columns to 1 column based on screen size.
- **Smart Toggle**: Features a rotating `+` icon that transforms into an `x` (rotated plus) when active.
- **Real-time Search**: A pill-shaped search bar that filters questions instantly with layout animations.
- **Independent Expansion**: Each card grows independently to reveal its answer without stretching its neighbors (once fixed).

## Why this choice?
This layout provides the highest information density while maintaining a premium, "technical" aesthetic. It feels more like a modern software dashboard than a generic list of questions, aligning perfectly with the **Oryonix AI** brand.
